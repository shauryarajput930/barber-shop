## 🚀 Getting Started

### **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### **Installation**
1. Clone or download the project
2. Navigate to the project directory
3. Open `index.html` in your browser
4. Or run a local server:
   ```bash
   python -m http.server 8000
   ```
5. Visit `http://localhost:8000`

## 🎨 Customization

### **Colors**
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #c9a55c;    /* Gold accent */
    --primary-hover: #b08d45;    /* Darker gold */
    --bg-dark: #121212;          /* Dark background */
    --text-light: #ffffff;        /* White text */
}
```

### **Content**
- Update text in `index.html`
- Replace images in `assets/` folder
- Modify contact information

### **Services**
Edit pricing cards in the services section:
```html
<div class="pricing-card">
    <div class="card-header">
        <h3>Service Name</h3>
        <div class="price">$XX</div>
    </div>
    <ul class="card-features">
        <li>✓ Feature 1</li>
        <li>✓ Feature 2</li>
    </ul>
    <button class="card-btn">Book Now</button>
</div>
```
